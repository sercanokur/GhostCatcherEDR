# Security

If you believe you have found a security vulnerability in GhostCatcher, please **do not** open a public GitHub issue for it.

**Preferred:** use [GitHub Security Advisories](https://github.com/sercanokur/GhostCatcherEDR/security/advisories/new) (private reporting) if the feature is enabled on the repository.

**Alternative:** contact the maintainer at the email address on their GitHub profile or in recent signed git tags, and allow a reasonable time for a fix before any public disclosure.

Include: affected version or commit, reproduction steps, and impact summary when possible.
